## Énfasis de texto
* Para nosotros utilizar un texto en **negrita** se utilizan dos "**".
* Para nosotros utilizar un texto en *cursiva* se utilizan un "*".
* Para nosotros ~~tachar~~ un texto se utilizan dos "~~".
